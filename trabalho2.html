Curso de JavaScript

O curso de JavaScript ensina desde o básico até conceitos avançados para desenvolver aplicações interativas na web.

Conteúdo do curso:

O curso de JavaScript ensina desde os conceitos mais básicos até técnicas avançadas, capacitando o aluno a desenvolver aplicações modernas, dinâmicas e interativas para a web.

Conteúdo do curso (clique nos tópicos para mais detalhes):

  • Introdução ao JavaScript
  • Manipulação de DOM
  • Funções e Objetos
  • Desenvolvimento de projetos
  • Introdução ao JavaScript
    Neste módulo, você aprenderá o que é JavaScript, sua importância no desenvolvimento web, como integrá-lo em páginas HTML e como executar seus primeiros scripts. Também será introduzido aos principais conceitos da linguagem, como variáveis, operadores, tipos de dados e estruturas de decisão.
  • Manipulação de DOM
    Aqui você aprenderá como interagir com os elementos HTML por meio do DOM (Document Object Model). Isso inclui como selecionar elementos da página, alterar conteúdos, adicionar ou remover classes, manipular atributos e criar efeitos dinâmicos que respondem às ações do usuário em tempo real.
  • Funções e Objetos
    Neste tópico, aprofundamos o uso de funções em JavaScript, incluindo funções anônimas, arrow functions e callbacks. Você também aprenderá a trabalhar com objetos, criar propriedades e métodos, além de entender conceitos como encapsulamento, herança e orientação a objetos com JavaScript moderno (ES6+).
  • Desenvolvimento de projetos
    Esta etapa do curso é totalmente prática. Você aplicará tudo o que aprendeu em projetos reais, como uma calculadora funcional, um gerenciador de tarefas (to-do list) e uma galeria de imagens com filtros. O objetivo é consolidar o conhecimento e criar um portfólio inicial com projetos interativos e bem estruturados.
